- **Problem**: API returning 0 tokens, all fields zeroed
- **Root cause**: `--raw` flag returns empty `tokens` array; without `--raw` data is in `data['rank']` with different field names
- **Fix**: Try `--raw` first, fallback to standard format, convert `rank` items to token format
- **Age calculation**: `age = now - creation_timestamp` (field was missing)
- **Field mapping**: `symbol` (not `sym`), `market_cap` (not `mcap`), `price_change_percent1h` (not `h1`)
